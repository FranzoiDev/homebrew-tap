# FranzoiDev Homebrew Tap

Homebrew formulae for [FranzoiDev](https://github.com/FranzoiDev) projects.

## Install

```bash
brew install franzoidev/tap/ai-usagebar
ai-usagebar   # start the menu bar app
```

## Formulae

- **[ai-usagebar](Formula/ai-usagebar.rb)** — native macOS menu bar app that
  surfaces AI plan usage. Builds from source, so the binary is signed by your
  own toolchain and isn't quarantined (no Gatekeeper prompt). Needs the Xcode
  toolchain to compile. See
  [ai-usagebar-macos](https://github.com/FranzoiDev/ai-usagebar-macos).
